# Encryption Algorithm RGR

## Консольное приложение на C++ для шифрования и дешифрования текста и файлов с использованием подключаемых криптографических алгоритмов.

## Реализованные алгоритмы
 * RC4
 * Шифр Виженера
 * XTEA
 * OTP
 * Шифр Хилла
 * Эль-Гамаль

## Каждый алгоритм реализован как отдельный плагин (.so) и подключается во время работы программы.

## Возможности
 * Генерация ключей
 * Шифрование текста
 * Дешифрование текста
 * Шифрование файлов
 * Дешифрование файлов
 * Работа с текстовыми и бинарными файлами

## Структура проекта
 * main.cpp — пользовательский интерфейс и меню
 * plugin_manager.* — загрузка и управление плагинами
 * crypto_interface.h — общий интерфейс алгоритмов
 * file_manager.* — чтение и запись файлов
 * base64_utils.* — кодирование Base64
 * *_plugin.cpp — реализации алгоритмов
 * plagins/ — динамические библиотеки алгоритмов

## Сборка
1) chmod +x build.sh
2) ./build.sh
## Запуск
1) ./main
2) Пароль по умолчанию
12345678
## Принцип работы

При запуске программа загружает все плагины из папки plagins, получает через dlopen и dlsym функции алгоритмов и предоставляет пользователю выбор алгоритма для шифрования или дешифрования данных.
